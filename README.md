# Thesis
Safe Reinforcement Learning for Real-time Microgrid Reconfiguration and Load Shedding Under Nonlinear Power Flow Constraints

# Libraries
## Panadapower
pandapower is used to build and simulate the electric network and solving AC powerflow equations. 

## Pandas
pandas is used to store bus layout (geodata).

## Numpy 
Handles state vectors.

## Plotly 
Interactive network rendering 

## Gym
Creates environment for the microgrid enabling plug and play with RL libraries
